## 📚 연산자

+, -, \*, / 과 같은 연산자에 대해서 알아보겠습니다.<br/>
다만, 너무 기초적인 부분이 아닌 알아두면 좋을 정보들을 정리 해보려 합니다.

- 피연산자(operand): 연산자가 연산을 수행하는 대상
- 단항(unary)연산자: 피연산자를 하나만 받는 연산자 ex) -x
- 이항(binary)연산자: 피연산자를 둘 받는 연산자 ex ) x + y

> 문자열 + 숫자형 => 문자열로 바뀐 후 연산이 됩니다.(그냥 문자열처럼 붙임)
> 그 외에 -, \*, / 는 숫자형으로 바뀐 후 연산이 진행됩니다.

[참고: 연산자의 순위](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Operator_Precedence)

### 단항 연산자 ➕

```js
// 단항 연산자로서 +는 Number()와 같은 효과를 갖습니다.
let a = true;
let b = '11';
console.log(+a); // 1
console.log(+a + +b); // 12
```

### ++value와 value++ 차이

++value는 값을 1 증가시키면서 1 증가시킨 값을 반환합니다.<br/>
value++는 값을 1 증가시키지만 그 자체의 반환값은 증가시키기 전의 값입니다.

```js
let a = 1;
let b = ++a; // a도 2가 되면서 b에도 2가 할당

let c = 1;
let d = c++; // c는 2가 되지만 d에는 1이 할당
```

### 쉼표 연산자

보기 드물지만 코드를 짧게 쓰기 위한 용도로 가끔 볼 수 있습니다.<br/>
쉼표에서의 마지막 값만 선택되어집니다.

```js
let a = (1 + 2, 3 + 4); // a엔 7이 할당됩니다.

// ,는 연산자로서 우선 순위가 굉장히 낮기 때문에 위의 식에서 괄호가 없다면
// 아래와 같은 결과가 나옵니다.
let b = 1 + 2, 3 + 4; // b엔 3이 할당됩니다.

// 사용예시
for (a = 1, b = 3, c = a * b; a < 10; a++) {
 ...
}
```

### null vs 0

`>, <, =<, >=` 와 같은 비교 연산자가 null에 쓰일 때는, null을 숫자형으로 변환되어 0이 되기 때문에 아래와 같은 다소 독특한 결과가 나옵니다.

```js
alert(null > 0); // (1) false
alert(null == 0); // (2) false
alert(null >= 0); // (3) true
```

### 조건부 연산자 ❓

조건부 연산자 ? 를 아래와 같이 if, if else, else와 같은 기능으로 표현할 수 있습니다.<br/>
어찌보면 코드가 짧아져 더 깔끔해보이지만, 가동성이 떨어지므로 복잡한 조건문에선 if, else의 표현이 더 좋습니다.

```js
let age = prompt('나이를 입력해주세요.', 18);

let message =
  age < 3
    ? '아기야 안녕?'
    : age < 18
    ? '안녕!'
    : age < 100
    ? '환영합니다!'
    : '나이가 아주 많으시거나, 나이가 아닌 값을 입력 하셨군요!';

alert(message);
```

### 논리 연산자

#### ||와 &&

논리 연산자에서 반드시 기억할 것은 `||`은 단순히 `또는(or)`이 아니고 `&&`도 단순히 `그리고(and)`가 아니라는 점입니다.

- || : 값들 중 truthy한 값을 찾게 되면 바로 반환합니다. 끝까지 truthy한 값이 없다면 맨 끝값을 반환합니다.
- && : 값들 중 falsy한 값을 찾게 되면 바로 반환합니다. 끝까지 falsy한 값이 없다면 맨 끝값을 반환합니다.

단순히 ||와 &&를 `또는`, `그리고`로 알고 있는 것과 위와 같이 알고 있는 것의 차이는 굉장히 큽니다.<br/>
이런 식으로 활용이 가능합니다.

```js
// 단락 평가
true || alert('not printed'); // 첫 값이 true이므로 뒤에까지 가지 않습니다.
false || alert('printed'); // 첫 값이 false이므로 뒤에도 확인하기에 alert가 실행됩니다.
```

#### !(NOT)

NOT 즉, 해당 값을 불린형으로 변환 후 그 불린값의 `역`을 반환합니다.

```js
let a = 1;
console.log(!a); // false

// !를 2개 연달아 붙여 !!를 사용함으로써 Boolean()과 같은 효과를 냅니다.
console.log(!!a); // true
```

#### ?? 연산자(nullish 병합 연산자)

최근에 추가된 문법으로 위의 논리 연산자(특히, ||)와 아주 유사하지만 조금은 다른 연산자입니다.<br/>
앞의 값이 `undefined`도 아니고 `null`도 아니면 그 값을 반환합니다.<br/>
즉, `값이 할당된 변수`를 빠르게 찾을 수 있습니다.

```js
let a = undefined ?? null ?? 1; // a는 1이 할당됩니다.
```

위의 ||와 &&처럼 `undefined, null이 아닌 값`을 찾지못하면 맨 끝의 값을 반환합니다.

> 아직 보완할 부분들이 있어, 괄호 없이 ||나 &&를 같이 쓰게 되면 생각한 연산이 안될 수 있습니다.
